# 🧾 Billing System (C++)

A simple console-based Billing System developed in C++ that generates customer invoices with GST and discount calculation.

## Features
- Add multiple items
- GST (18%) calculation
- Discount calculation
- Invoice generation
- Amount in words
- Invoice generation in .txt

## Technologies
- C++
- Code::Blocks
